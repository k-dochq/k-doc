## 1. 홈 화면

🔗 https://k-doc.kr/ko/v2/main

### 빠른 메뉴 (부위 카테고리 아이콘)
1. **카테고리 아이콘 — ON 상태 배경 원** — 연핑크(`#FEEFFF`) → 연보라(`#F1EEFF`)
2. **카테고리 아이콘 — ON 상태 장식 선(stroke)** — 연핑크(`#F8ADFF`) → 연보라(`#A9A5FF`)
3. **카테고리 아이콘 — ON 상태 주요 선/면 색** — 핑크 → 보라
4. **카테고리 아이콘 — OFF 상태** (구버전 아이콘) — 핑크(`#DA47EF`) → 보라(`#7657FF`)

### 이벤트 배너
5. **배너 하단 페이지네이션 점 — 활성** — 핑크 → 보라
6. **배너 하단 페이지네이션 점 — 비활성** — 연핑크 → 연보라

### 인기 리뷰 섹션
7. **슬라이드 진행 바 — 배경(비활성 구간)** — 연핑크 → 연보라
8. **슬라이드 진행 바 — 활성 구간** — 핑크 → 보라

### 검색 바
9. **검색 아이콘** — 핑크 → 보라
